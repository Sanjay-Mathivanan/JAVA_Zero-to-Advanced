##  Software Tools (Platform-wise Setup)

### For Windows 

* **Java JDK (Oracle / OpenJDK)** – Install and set `JAVA_HOME`
* **IntelliJ IDEA / VS Code** – Recommended IDEs for Java development
* **Command Prompt / PowerShell** – To compile and run Java programs
* **Git Bash** – For version control operations

---

### For macOS (Apple Devices)

* **Java JDK (OpenJDK / Oracle JDK)** – Installed via Homebrew or manual setup
* **IntelliJ IDEA / VS Code** – Smooth and optimized for macOS
* **Terminal (Zsh/Bash)** – Default shell for running Java programs
* **Homebrew** – Package manager to install Java and tools easily

---

### For Apple Silicon (M1/M2/M3 Macs)

* **ARM-compatible JDK** (like OpenJDK or Azul Zulu)
* **IntelliJ IDEA (Apple Silicon version)** – Better performance
* **VS Code (ARM build)** – Lightweight and fast
* **Terminal + Rosetta (if needed)** – For compatibility with older tools

---

### 🔧 Common Tools (All Platforms)

* **Git & GitHub** – Version control and project hosting
* **Debugger Tools** – For error tracking and code analysis
* **Browser** – For documentation and learning resources

---

⚡ Choosing the right tools based on your system ensures better performance and a smoother Java development experience.
