# File Handling in Java

This module covers Java File I/O stream architectures, File metadata management, Object Serialization, and modern Java NIO.2 APIs.

## Lessons Map

### 1. I/O Stream Foundations
* [Introduction to File Handling](01_Introduction_to_File_Handling.md): Stream abstractions, Byte Streams vs. Character Streams, and I/O class hierarchies.
* [File Class Basics](02_File_Class_Basics.md): `java.io.File` API, creating/deleting files and directories, and inspecting file metadata.

### 2. Binary and Character Processing
* [Byte Streams](03_Byte_Streams.md): `FileInputStream` and `FileOutputStream` operations, binary file copying, and array buffer reads.
* [Character Streams](04_Character_Streams.md): `FileReader` and `FileWriter` operations, and explicit UTF-8 character encoding.

### 3. Serialization and Modern NIO.2
* [Serialization and Deserialization](05_Serialization_and_Deserialization.md): Object streams, `Serializable` marker interface, `transient` fields, and `serialVersionUID` versioning.
* [NIO.2: Files and Paths](06_NIO_Files_and_Paths.md): `Path`, `Paths`, `Files` helper utilities, and recursive stream directory walking.
