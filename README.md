# Java: Zero to Advanced

This repository documents a comprehensive journey of mastering Java from scratch to advanced engineering levels, with a strong focus on **deep, low-level understanding rather than rote memorization**.

---

## Roadmap & Coverage

Here is the directory map of the learning path:

1. **[01. Setup](01_SETUP/)**: Installing Java, configuring environment paths, running your first Java compilation via CLI.
2. **[02. Introduction](02_Introduction/)**: Core concepts of compilation vs execution, JVM, JRE, JDK, bytecode, and basic syntax structure.
3. **[03. Function Design](03_function_design/)**: Variables, primitive types, non-primitives, stack vs heap, parameters, arguments, return types, and call stack operations.
4. **[04. Control Flow Statements](04_control-flow-statements/)**: Conditional branching, loops (`for`, `while`, `do-while`), jump statements (`break`, `continue`, `return`), and flow charts.
5. **[05. Building Blocks of Java](05_building-blocks-of-java/)**: Classes, objects, memory allocation (stack vs heap), methods, constructors, instantiation, class members vs instance members, and the `this` and `super` keywords.
6. **[06. OOP Pillars](06_OOPs-In-Java/)**: Composition vs Inheritance, Encapsulation with validators, Single/Multilevel/Hierarchical inheritance, Compile-Time/Runtime Polymorphism, Upcasting, Downcasting, and Abstract Classes.
7. **[07. Naming Conventions & Packages](07_Naming-Conventions-and-Packages-in-Java/)**: Standard coding naming rules and package structuring.
8. **[08. Abstract Features of Java](08_Abstract-Features-of-Java/)**: Deeper focus on interfaces, static vs default methods, and abstraction levels.
9. **[09. Advanced Java Class Concepts](09_Advanced-Java-Class-Concepts/)**: Inner classes, local classes, anonymous classes, lambda expressions, and functional interfaces.
10. **[10. Collection Framework](10_Collection-Framework-in-Java/)**: Lists, Sets, Maps, Queues, Iterators, and algorithm complex analysis.
11. **[Exception and Buffer](11_Exception-and-Buffer/)**: Checked vs. Unchecked exceptions, try-catch-finally, custom Exceptions, try-with-resources, StringBuffer, StringBuilder, and BufferedReader/Writer.

---

## Learning Method

Each topic follows a structured learning loop:
$$\text{Concept Theory} \longrightarrow \text{Low-Level Memory Analysis} \longrightarrow \text{Code Implementation} \longrightarrow \text{Practice Challenges}$$

---

## Repository Goal

All contents, diagrams, and exercises are explained in clean, beginner-friendly language, focusing on **why** things behave the way they do in memory.

---

*Building consistency. Building depth. Building mastery.*
