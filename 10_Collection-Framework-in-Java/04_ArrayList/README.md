# ArrayList in Java

This directory covers the `ArrayList` class, which is a dynamically resizable array implementation of the `List` interface.

1. [ArrayList Basics](01_ArrayList-Basics.md): Syntax, characteristics, and constructors.
2. [ArrayList Methods - Part 1](02_ArrayList-Methods-Part1.md): Appending, inserting, getting, and updating elements.
3. [ArrayList Methods - Part 2](03_ArrayList-Methods-Part2.md): Removing elements, clear, array conversions, and capacity trim operations.
4. [ArrayList Traversal and Sorting](04_ArrayList-Traversal-and-Sorting.md): Iterating using Loops, Iterators, and ListIterators, sorting, and binary search rules.
5. [ArrayList Internal Workings](05_ArrayList-Internal-Workings.md): Array allocations on Heap, capacity expansion mathematical formula, and performance complexities.
