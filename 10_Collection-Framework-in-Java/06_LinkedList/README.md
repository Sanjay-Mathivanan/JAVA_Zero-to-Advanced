# LinkedList in Java

This directory covers the `LinkedList` class, which is a doubly linked list implementation of the `List` and `Deque` interfaces.

3. [LinkedList Iteration](03_LinkedList-Iteration.md): Sequential traversal, safe removals, and the quadratic complexity warning on index-looping.
4. [LinkedList Internal Workings](04_LinkedList-Internal-Workings.md): Pointers connecting, head/tail pointers caching, and node object instantiation.
5. [LinkedList vs. ArrayList](05_LinkedList-vs-ArrayList.md): Direct performance, size, and complexity comparison matrices.
6. [LinkedList Interview Questions](06_LinkedList-Interview-Questions.md): Standard beginner, intermediate, and advanced FAQs.
