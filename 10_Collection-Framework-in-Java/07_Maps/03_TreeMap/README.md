# TreeMap in Java

This directory covers `TreeMap`, which is a Red-Black tree-based implementation of the `NavigableMap` and `SortedMap` interfaces, sorting keys automatically.

## Lessons Map

1. [TreeMap Basics and Operations](01_TreeMap-Basics-and-Operations.md): Auto-sorting, Comparators, and NavigableMap range query APIs.
2. [TreeMap Internal Workings](02_TreeMap-Internal-Workings-and-Comparison.md): Red-Black tree self-balancing, BST depth invariants, and comparisons.
