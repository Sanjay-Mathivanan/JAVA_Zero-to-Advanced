# Ticket Booking System Simulation

## Overview
This mini-project demonstrates the First-In, First-Out (FIFO) principle of standard queues in a ticket booking scenario.

## Simulation Flow
1. Customers line up in sequence to purchase tickets.
2. Available tickets are limited.
3. The system processes the customers at the head of the queue first.
4. Late-joining customers are notified if tickets sell out.

## Class Used
* **`ArrayDeque`**: Serves as the FIFO queue implementation.
